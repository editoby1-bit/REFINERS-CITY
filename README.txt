REFINERS CITY ATTENDANCE APP

WHAT IS INCLUDED
- Full browser-based attendance and member database app
- Church Admin, Ordained Pastor, and G12 Pastor login roles
- Member registration with name, phone and address
- Sunday 1st, 2nd, 3rd, 4th service presets
- Midweek service preset
- Custom service / conference / activity creation
- Admin-only attendance marking
- G12 pastors can assign members to their own class
- A member cannot belong to two G12 classes
- Read-only dashboards for pastors
- Church logo placed in sign-in page and main app header
- Data saved in browser local storage

HOW TO RUN
1. Extract the ZIP.
2. Open index.html in a browser.
3. Use the demo logins below or create new accounts.
